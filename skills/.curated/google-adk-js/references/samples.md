# ADK TypeScript Samples

## Repository
- https://github.com/google/adk-samples

## Layout
The TypeScript samples live under:
```
adk-samples/typescript
  agents/
    <agent-name>/
      README.md
```

## Typical workflow
1. Clone the repo: `git clone https://github.com/google/adk-samples.git`
2. Go to `adk-samples/typescript`.
3. Pick an agent in `agents/` and follow its README.
4. Copy patterns into your project (agent structure, tools, env vars).
